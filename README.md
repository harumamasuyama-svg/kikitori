# キキトリ

話した内容や文字起こしを、日記・議事録・要約の形式で清書する試作Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 日記・議事録・要約のモードを選びます。
3. 「元の内容」に文章を入力します。
4. 「選んだモードで清書」を押します。
5. 必要に応じてコピー、画像、PDF、Wordで出力します。

## GitHub Pages

GitHub Pagesで公開する場合は、リポジトリの Settings → Pages から、Branch を `main`、Folder を `/ (root)` に設定します。

## 注意

現在の版はブラウザ内だけで動く簡易版です。本格的な要約・議事録化には、OpenAI APIなどのAI連携用サーバーが必要です。
