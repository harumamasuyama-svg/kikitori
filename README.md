# キキトリ

スマホのブラウザで使える、音声入力から日記を清書する試作アプリです。

## GitHub Pagesで使う場合

index.html を公開すれば、音声入力、日記の簡易清書、コピー、画像、PDF、Word出力が使えます。
GitHub Pages上ではAPIキーを安全に置けないため、AI清書は使わず簡易清書になります。

## ローカルAI連携版

OpenAI APIキーをブラウザに書かず、パソコン上の小さなサーバーだけに持たせる使い方です。

1. PowerShellでこのフォルダを開きます。
2. APIキーをそのPowerShellだけに設定します。

```powershell
$env:OPENAI_API_KEY="sk-あなたのAPIキー"
```

3. 起動します。

```powershell
node server.js
```

4. ブラウザで開きます。

```text
http://127.0.0.1:4173/index.html?v=13
```

ポートが使用中の場合は、次のように別ポートで起動できます。

```powershell
$env:PORT=4174
node server.js
```

## AI清書でできること

- 日記: 話し言葉を自然な日記に整えます。
- 議事録: 会議の文字起こしを、概要、議題、要点、決定事項、課題、次の対応に整理します。
- 要約: 長い文章を短く分かりやすくまとめます。

## 注意

APIキーはGitHubにアップロードしないでください。.env は無視されるようにしていますが、キーはPowerShellで一時的に設定する方法が安全です。

## iPhoneからローカルAI連携版を試す場合

WindowsとiPhoneを同じWi-Fiにつなぎ、サーバーを次のように起動します。

```powershell
$env:HOST="0.0.0.0"
node server.js
```

その後、WindowsのIPアドレスを使って iPhone のSafariで開きます。例: `http://192.168.1.10:4173/`

外出先や他人のWi-Fiでは使わず、自分の管理しているネットワーク内で使ってください。

## かんたん起動

Windowsでは `start-kikitori-ai.bat` をダブルクリックすると、APIキーを貼り付けるだけで起動できます。入力したキーはファイルに保存されません。

## 録音ファイルの文字起こし

AI連携版では、mp3、m4a、wav、webmなどの録音ファイルを選んで文字起こしできます。OpenAI APIの音声アップロードにはサイズ制限があるため、まずは25MB以内のファイルで試してください。2時間程度の録音は、m4aなどで25MB以内ならそのまま使える場合があります。超える場合は分割または圧縮が必要です。
